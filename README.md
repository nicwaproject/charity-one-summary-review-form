# charity-one-summary-review-form
clean &amp; minimal web based version of the summary review&amp; change update form for Charity One Insurance Agency.
Instalation & deployment Step (1. Upload all project files to your hosting) (2. configure the submisson endpoint, update this line const endpointURL = "YOUR_ENDPOINT_HERE";
No backend is included in this repo - submission handling is perfomed by your internal system.
File uploads use the standard <input type="file"> element.
The form is mobile-responsive and WCAG friendly.
